# Serie RAG - Les 12 architectures RAG

Serie de 3 presentations pedagogiques sur les architectures RAG (Retrieval-Augmented Generation).

Audience cible : managers, recruteurs, profils metier non-developpeurs.
Niveau : fondamental a intermediaire (aucun code, analogies concretes).
Format : presentations Gamma, 24 slides chacune, ~12-18 minutes de lecture.

## Pourquoi cette serie ?

Le RAG est la technique cle qui permet aux LLMs d'acceder a des connaissances recentes ou privees
sans retrainement. Comprendre les 12 architectures permet de choisir la bonne selon le contexte :
volume documentaire, fiabilite requise, couts, complexite d'implementation.

Cette serie est liee au projet Lyla-OS (architecture RAG hybride personnelle de Peter).
